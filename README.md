# DesignPattern
Jungle设计模式系列

## How to compile
### Windows
```{.line-numbers}
mkdir build
cd build
cmake -G "MinGW Makefiles" ../
make
```
### Linux
```{.line-numbers}
mkdir build
cd build
cmake ../
make
```
